# AFL Betting Model API

FastAPI service that predicts AFL player disposals over/under probabilities.

## Features
- Predicts three thresholds: over 19.5, 24.5, 29.5 disposals
- Context features: opponent strength, venue average, bye indicator, rolling history
- Model: RandomForest trained on 2020-2025 data (54,589 player-games)

## Local Development

```bash
# Create virtualenv
python -m venv venv
source venv/bin/activate  # Windows: venv\\Scripts\\activate
pip install -r requirements.txt

# Run API
uvicorn src.api:app --reload
```

API available at http://localhost:8000/docs (Swagger UI).

## Deployment

### Railway
```bash
# Install Railway CLI and link project
railway init
# Set environment if needed (none required)
railway up
```

### Render
Create a Web Service pointing to this repo, build command: `pip install -r requirements.txt`, start command: `uvicorn src.api:app --host 0.0.0.0 --port $PORT`.

## Model Training
Models are generated by `src/backtest_context.py`. The script outputs `models/context/models_context.pkl` and `models/context/features.pkl`.

## Notes
- Training data: 2020-2025 (after 2026 season started)
- Features computed: rolling disposals (5/10/20 games), standard deviation, previous game.
- Context: historical opponent averages (offensive and defensive), venue average disposals, bye indicator, home/away.
- Weather and betting odds integration experiments did not improve performance.
